Timestamp Microservice

Timestamp API

This API returns UTC time and Unix timestamp (milliseconds).

Details

GET /api/[date or timestamp]

Replace [date or timestamp] with a valid date or a timestamp to get a JSON object as response.

E.g: "/api/2002-02-02" or "/api/1012608000000".

Request
[Base URL]/api/2002-02-02 or [Base URL]/api/1012608000000
Response
{"unix":1012608000000, "utc":"Sat, 02 Feb 2002 00:00:00 GMT"}
